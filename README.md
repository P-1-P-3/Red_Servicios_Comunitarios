# RED_SERVICIOS_COMUNITARIOS 🤝

## 📝 1. Descripción del Proyecto
Este proyecto consiste en una plataforma digital y red de servicios comunitarios diseñada para conectar a los ciudadanos con soluciones, comercios y servicios clave de su entorno local, optimizando la gestión, la visibilidad de los emprendimientos y la comunicación comunitaria.

## 📐 2. Arquitectura General del Sistema
El backend está desarrollado bajo una arquitectura orientada a servicios en Node.js, siguiendo el patrón de diseño **Ruta - Controlador - Repositorio**:
* **Routes (Rutas):** Definen los puntos de entrada (endpoints) de la API.
* **Controllers (Controladores):** Manejan la lógica de negocio y las solicitudes de los usuarios (como la gestión del carrito de compras).
* **Repositories (Repositorios):** Se encargan de la comunicación directa y consultas a la base de datos.

## 🚀 3. Tecnologías Utilizadas
* **Entorno de Ejecución:** Node.js
* **Framework Web:** Express.js
* **Lenguaje:** JavaScript (ES6+)
* **Pruebas (Testing):** Jest & Babel
* **Documentación:** Swagger (API) y JSDoc (Código)
* **Control de Versiones:** Git & GitHub
